# 图片文件说明

## 必需图片

### logo.png
- **用途**: 项目Logo，显示在导航栏和Hero区域
- **建议尺寸**: 120×120px 或更高分辨率
- **格式**: PNG（支持透明背景）或 JPG

### arch.png
- **用途**: 方法/架构图，显示在Method章节
- **建议尺寸**: 宽度至少 800px
- **格式**: PNG 或 JPG

### results.png
- **用途**: 实验结果图表，显示在Results章节
- **建议尺寸**: 宽度至少 800px
- **格式**: PNG 或 JPG

## 可选图片

您可以根据论文内容添加其他配图，例如：
- 方法示意图
- 实验结果对比图
- 数据集可视化图
- 等等

## 图片要求

- **格式**: PNG, JPG, 或 WebP
- **分辨率**: 建议使用高分辨率图片以确保清晰度
- **文件大小**: 建议每张图片不超过 2MB
- **优化**: 可以使用图片压缩工具优化文件大小

## 占位符

如果图片文件不存在，页面会自动隐藏图片元素（通过 `onerror` 处理），不会影响页面布局。
